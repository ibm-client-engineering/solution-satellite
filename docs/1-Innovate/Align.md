## Vision
The purpose of this document is to provide a step-by-step guide for deploying an application on a satellite location on IBM and AWS using IBM Cloud Console, Terraform, and Helmcharts. The solution aims to address the business need to host a business application on different cloud environments to provide a scalable, secure and efficient process to deploy and run applications.

This is a living document that is subject to change and evolution as IBM Client Engineering co-creates this solution with our customer.
