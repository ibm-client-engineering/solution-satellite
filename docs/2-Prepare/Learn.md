## Learning Resources

:::tip

Prior to building out Satellite locations on IBM Cloud and AWS, you need to have a basic understanding of IBM Satellite, Terraform, Helm charts, and the Legend application.

:::

- [IBM Cloud Satellite](https://cloud.ibm.com/docs/satellite?topic=satellite-getting-started)
This documentation wiill provide an overview for the different components, capabilitie, and pre-requisites to start familiarizing one's self with working with IBM Cloud Satellite.

- [Getting started with Terraform](https://developer.hashicorp.com/terraform/tutorials)
Overview on the fundamentals of Terrafrom and how it works along with various tutorials.

- [Satellite github](https://github.com/terraform-ibm-modules/terraform-ibm-satellite/tree/main)
Documentation and resources on how to write terraform scripts to create satellite locations through various cloud providers.

- [Legend documentation] (https://legend.finos.org/docs/overview/legend-overview)
Comprehensive website explaining the purpose of the Legend app and how to install it.
