## Learning Resources

:::tip

Prior to building out Satellite locations on IBM Cloud and AWS and deploying Legend to them, you need to have a basic understanding of IBM Satellite, Terraform, Helm charts, and the Legend application.

:::

- [IBM Cloud Satellite](https://cloud.ibm.com/docs/satellite?topic=satellite-getting-started)
This documentation will provide an overview of the different components, capabilities, and pre-requisites to start familiarizing oneself self with working with IBM Cloud Satellite
- [Getting started with Terraform](https://developer.hashicorp.com/terraform/tutorials)
Overview of the fundamentals of Terraform and how it works, along with various tutorials.
- [Legend documentation] (https://legend.finos.org/docs/overview/legend-overview)
Comprehensive website explaining the purpose of the Legend app and how to install it.
