## Deploy Legend to Satellite Locations

1. Navigate to appropriate satellite env
2. Navigate to sidebar and select 'Satellite' -> 'Clusters and Cluster groups' -> 'Cluster groups' -> 'Create Cluster Group'
3. Name the Cluster Group and assign the relevant satellite locations
4. Within Satellite window, go to 'Configurations' and 'Create Configuration'
5. Select 'Direct Upload', and fill out all appropriate fields:
    * <b> Configuration: </b> Populate Name and location
    * <b> Version: </b> Populate version and upload yaml config for application
    * <b> Subscription: </b> Fill out name and assign the appropriate cluster groups (groups crefated in Step 2)
    * <b> Summary: </b> Review and click 'Craete'
